# SemOps2 User Narratives

This folder contains end-to-end usage narratives for SemOps2. Each narrative focuses on a specific flow and includes:

- Purpose and context
- Actors and preconditions
- Step-by-step narrative
- Mermaid diagram (flowchart or sequence diagram)
- Related BDD scenarios and links

Use these narratives to align on user intent before implementation.
